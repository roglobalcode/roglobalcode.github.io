Cara Melihat Password WiFi di Android dan Komputer

By Putri Pujiastuti
April 27, 2023

WiFi menjadi salah satu teknologi yang sangat penting dalam kehidupan sehari-hari. Dengan WiFi, kita bisa terhubung ke internet tanpa perlu menggunakan kabel. Namun, terkadang kita lupa password WiFi yang sudah tersimpan di perangkat. Jangan khawatir, di artikel ini kita akan membahas cara melihat password WiFi di Android dan komputer.

Cara Melihat Password WiFi di Android

1. Menggunakan Aplikasi WiFi Key Recovery

Langkah pertama yang bisa dilakukan adalah dengan menggunakan aplikasi WiFi Key Recovery. Aplikasi ini dapat membantu kita untuk melihat password WiFi yang tersimpan di perangkat Android.

Berikut adalah langkah-langkahnya:

  1. Download dan install aplikasi WiFi Key Recovery dari Google Play Store.
  2. Buka aplikasi tersebut dan tunggu hingga proses scanning selesai.
  3. Pilih jaringan WiFi yang ingin dilihat passwordnya.
  4. Aplikasi akan menampilkan password WiFi tersebut.

2. Menggunakan Command Prompt

Cara selanjutnya adalah dengan menggunakan Command Prompt. Namun, cara ini hanya bisa dilakukan jika perangkat Android sudah di-root.

Berikut adalah langkah-langkahnya:

  1. Download dan install aplikasi Terminal Emulator dari Google Play Store.
  2. Buka aplikasi tersebut dan ketikkan perintah "su" untuk mendapatkan akses root.
  3. Ketikkan perintah "cd /data/misc/wifi" untuk masuk ke direktori WiFi.
  4. Ketikkan perintah "cat wpa_supplicant.conf" untuk melihat password WiFi yang tersimpan.

Cara Melihat Password WiFi di Komputer

1. Menggunakan Command Prompt

Cara pertama untuk melihat password WiFi di komputer adalah dengan menggunakan Command Prompt.

Berikut adalah langkah-langkahnya:

  1. Buka Command Prompt dengan cara menekan tombol Windows + R dan ketikkan "cmd".
  2. Ketikkan perintah "netsh wlan show profile" untuk melihat profil WiFi yang tersimpan.
  3. Pilih profil WiFi yang ingin dilihat passwordnya dan ketikkan perintah "netsh wlan show profile [nama profil] key=clear".
  4. Password WiFi akan ditampilkan di bagian "Key Content".

2. Menggunakan Software Third-Party

Cara kedua adalah dengan menggunakan software third-party seperti WirelessKeyView atau Wi-Fi Password Recovery. Software ini dapat membantu kita untuk melihat password WiFi yang tersimpan di komputer.

Berikut adalah langkah-langkahnya:

  1. Download dan install software WirelessKeyView atau Wi-Fi Password Recovery.
  2. Buka software tersebut dan tunggu hingga proses scanning selesai.
  3. Pilih jaringan WiFi yang ingin dilihat passwordnya.
  4. Software akan menampilkan password WiFi tersebut.

Kesimpulan

Melihat password WiFi yang tersimpan di perangkat Android dan komputer sebenarnya cukup mudah dilakukan. Dalam artikel ini, kita telah membahas cara melihat password WiFi di Android dan komputer dengan menggunakan aplikasi, Command Prompt, dan software third-party. Namun, kita juga harus selalu ingat untuk tidak menyebarluaskan password WiFi kita agar tidak disalahgunakan oleh orang yang tidak bertanggung jawab.

FAQs

  1. Apakah semua perangkat Android dapat menggunakan aplikasi WiFi Key Recovery?
  • Tidak, hanya perangkat yang sudah di-root saja yang dapat menggunakan aplikasi WiFi Key Recovery.
  1. Apakah cara melihat password WiFi di komputer dapat dilakukan tanpa menggunakan Command Prompt?
  • Ada beberapa software third-party yang dapat membantu kita untuk melihat password WiFi tanpa menggunakan Command Prompt.
  1. Apa yang harus dilakukan jika tidak bisa melihat password WiFi dengan cara-cara yang telah dijelaskan?
  • Coba untuk menghubungi pemilik jaringan WiFi dan meminta passwordnya.
  1. Apakah aman menggunakan software third-party untuk melihat password WiFi?
  • Ya, asalkan software tersebut diunduh dari sumber yang terpercaya.
  1. Apakah bisa melihat password WiFi yang sudah pernah terhubung di perangkat Android atau komputer?
  • Ya, password WiFi yang pernah terhubung akan tersimpan di perangkat Android atau komputer dan bisa dilihat menggunakan cara-cara yang telah dijelaskan di artikel ini.
